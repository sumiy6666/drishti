# Frontend (React) - Enhanced

## New features
- React Router
- Material UI components
- Resume upload integration (presigned S3 flow)
- Email verification & password reset UI
- Messages UI

## Setup
- Create .env with REACT_APP_API if needed (default: http://localhost:5000)
- Install dependencies:
  ```
  cd frontend
  npm install
  ```
- Run:
  ```
  npm start
  ```
